# Yusuf - AI Mühendisi Kişisel Web Sitesi

Erciyes Üniversitesi'nde AI ile uğraşan Bilgisayar Mühendisi kişisel web sitesi.

## Özellikler
- Modern ve responsive tasarım
- Smooth scroll navigasyon
- Projeler ve yetenekler bölümü
- İletişim bilgileri

## GitHub Pages ile Yayınlama

1. Bu repoyu GitHub'a yükleyin:
   ```bash
   git init
   git add .
   git commit -m "İlk commit"
   git branch -M main
   git remote add origin https://github.com/kullanici-adi/personal-website.git
   git push -u origin main
   ```

2. GitHub'da repository ayarlarına gidin
3. "Pages" sekmesine tıklayın
4. "Source" olarak "Deploy from a branch" seçin
5. "Branch" olarak "main" ve "/ (root)" seçin
6. "Save" butonuna tıklayın

Web siteniz `https://kullanici-adi.github.io/personal-website/` adresinde yayınlanacak.

## Düzenleme

- İçerikleri `index.html` dosyasından düzenleyin
- Tasarımı `styles.css` dosyasından özelleştirin
- İnteraktif özellikleri `script.js` dosyasından geliştirin
